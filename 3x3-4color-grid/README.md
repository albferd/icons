# What
These are the same icons as 3x3-binary-grid; 3x3 grids of dots arranged in an svg.
In this case, these are SVGs which instead are base-4, in CMYK color (black is first)

Use them for whatever you want.

## Important thing
Only the first 2048 svgs have been generated; because the total amount of them would be 4^9 aka **262144** 😭 (I don't even think you can upload that many files lmao)
Use generate.py if you want a specific one
